# Maven Installation

```xml
<dependency>
    <groupId>com.apurebase</groupId>
    <artifactId>arkenv</artifactId>
    <version>${arkenv_version}</version>
</dependency>

<repositories>
    <repository>
        <id>jcenter</id>
        <url>https://jcenter.bintray.com/</url>
    </repository>
</repositories>
```
